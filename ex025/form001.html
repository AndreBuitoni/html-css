Informações sobre Formulários

Diferença entre os atributos NAME e ID

O atributo NAME é útil para HTML e PHP. Por outro lado, o atributo ID é mais útil para JavaScript

Métodos GET e POST

O método GET, que é o método padrão, pode ser utilizado quando você não está coletando dados sensíveis do usuário, por exemplo nome, peso, altura e etc. Quando for coletar dados sensíveis, por exemplo dados de cartão de crédito ou endereços, você deve utilizar o método POST, que apresenta maior segurança.

O que é input? E como usar?

Input é uma etiqueta/tag que cria um campo digitável onde o usuário pode colocar dados para enviar para o site. Para utilizar um input você deve informar os atributos type e value. No nosso exemplo, utilizamos type="submit" e value="Enviar".

O que é label? E como usar?

Label é uma etiqueta/tag que conecta o texto de um campo ao quadro do input. No nosso primeiro formulário, ali em cima, eu utilizei uma label para conectar "Nome:" com a caixa do input ao seu lado direito. Também utilizei uma segunda label para conectar "Sobrenome:" à caixa do outro input do formulário.
