Exemplo de Formulários usando os tipos RADIO e CHECKBOX

O Formulário abaixo é um exemplo do uso do botão tipo RADIO. Neste tipo de botão, o usuário só pode escolher uma alternativa. Para funcionar corretamente, os atributos name dos inputs deverão ser iguais para todas as opções. Use o atributo label para permitir que o usuário também possa clicar na palavra para selecionar o botão redondo.

Neste próximo Formulário, veja um exemplo do uso do botão tipo CHECKBOX. Neste tipo de botão, o usuário pode escolher várias alternativas. Os atributos name dos inputs podem ser diferentes para cada opção. Use o atributo label para permitir que o usuário também possa clicar na palavra para selecionar o botão quadrado.
