Estou Aprendendo a Criar Media Queries


O Objetivo deste exercício foi criar uma página de estilos CSS separada para o formato de impressão. Para testar, digite CTRL+P nesta página e observe que o layout de impressão é diferente do layout do website.


Na versão IMPRESSA do website, por exemplo, foi removida a barra de navegação, o texto deste artigo foi justificado, ao invés de alinhado a esquerda, e também foi adicionada uma frase extra abaixo do artigo. Para mostrar a frase no final utilizamos um comando chamado article::after diretamente na folha de estilos impressora.css

Na versão website utilizamos uma fonte sem serifa. Já na versão para impressão, utilizamos uma fonte serifada.

Nesta primeira aula, aprendemos que as Media Queries servem para modificar o código de acordo com o dispositivo que o usuário está utilizando. Geralmente, nós devemos fazer versões diferentes para celular, tablet, computador, televisão e mídia impressa
